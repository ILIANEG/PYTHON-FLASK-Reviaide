About this application

This web application allows one to add products to the system and allow people to access comments diredtly using system-generated QR code. You can access such QR code simply by adding product to the system, you can later access this items in "manage items" section.

Why this application?

There is no need to create a brand new page and set up server to collect reviews and feedback of your product and business. Your customers add comment or send feedback, and you just see them, it's that simple. This system is specificaly tailored for those who looking for a physical product solution, allowing your customers to access this system directly through QR code and with no registration. All of this ensures that as many people as possible will interact with your product. In future we also will add functinality to automatically answer questions using chat bots
